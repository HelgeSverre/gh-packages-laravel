# 📋 Record Management System (RMS)

> A smart, AI-powered document and record management platform built with Laravel.

## 🎯 What is RMS?

**Record Management System (RMS)** is a modern web application that helps you organize, store, and manage your documents and records efficiently. Think of it as your digital filing cabinet with superpowers.

## ✨ What Does This Project Do?

- **📁 Document Storage** - Upload and organize PDF files, images, and text documents
- **🤖 AI-Powered Analysis** - Automatically extract information from documents using AI
- **⚡ Smart Form Filling** - AI reads your documents and fills out forms for you
- **🔍 Easy Search** - Find any record quickly with intelligent search
- **👥 User Management** - Secure access control for teams
- **📊 Analytics** - Track and monitor your records

## 🚀 Key Features

### AI VIBE Button

Upload a document → Click "AI VIBE" → Watch as AI automatically fills out all the form fields based on your document content. No more manual data entry!

### Smart Organization

- Automatic categorization
- Tag suggestions
- Priority levels
- Status tracking

### Secure & Fast

- Role-based permissions
- Fast document processing (5-7 seconds)
- Responsive design for all devices

## 🛠️ Built With

- **Laravel 12** - Backend framework
- **Filament 3** - Admin panel
- **Livewire 3** - Dynamic interactions
- **OpenAI GPT** - AI document analysis
- **Tailwind CSS** - Modern styling

## 🎯 Perfect For

- **Businesses** managing contracts and documents
- **Legal firms** organizing case files
- **Healthcare** handling patient records
- **Education** managing student documents
- **Any organization** that deals with paperwork

## 💡 Why RMS?

Instead of spending hours manually entering data from documents, let AI do the work. Upload your document, click a button, and watch your forms get filled automatically with accurate information.

## Installation

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
```

BEFORE U START MAKE SURE TO FILL YOUR OWN KEY ON `.env` FILE

Start the App

```bash
composer run dev
```
