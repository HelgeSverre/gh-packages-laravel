## 🔍 What is laravel-metrics-matomo?

This software helps you get easy access to website traffic data without needing to open complex tools. It connects to Matomo, a web analytics service, and shows data like visits, page views, types of devices visitors use, and where they come from.

You can use it to build reports that help you understand your website’s audience. This tool works with Laravel, a popular way developers build websites. If you want to see your website’s stats in a simple, clear way, laravel-metrics-matomo is designed for that.

- In case of errors, check internet connection and API token validity.
